# 模块编辑器使用说明

## 快速开始

### 方法1：直接运行测试场景（推荐）
1. 打开 `module_editor_test.tscn`
2. 直接运行场景
3. 编辑器会自动初始化10个base_module到可用模块列表

### 方法2：在编辑器中设置
1. 打开 `module_editor_ui.tscn` 或 `module_editor_test.tscn`
2. 选择 `ModuleEditorUI` 节点
3. 在Inspector面板中：
   - 确保 `root_module_scene` 已设置为 `base_module.tscn`
   - `available_modules` 可以留空（运行时会自动填充）
   - 或者手动添加模块数据

## 手动设置 available_modules

如果需要手动设置，格式如下：

```
Array[Dictionary] = [
  {
    "module_scene": base_module.tscn (PackedScene),
    "count": 10 (int)
  }
]
```

## 使用方法

1. **查看模块列表**：右侧显示可用的模块，每个模块显示图标和数量（x数字）
2. **拖拽模块**：从右侧列表点击并拖拽模块到左侧预览区域
3. **吸附放置**：将模块拖到已有模块的上下左右位置，绿色表示可以放置，白色表示不能放置
4. **释放放置**：在绿色位置释放鼠标，模块会自动吸附到正确位置

## 保存和加载

- `get_module_data()`: 获取当前编辑的模块数据结构
- `save_module_data()`: 保存完整的模块结构
- `load_module_data(data)`: 加载保存的模块数据
